## Setup
1. Install dependencies <br>
`yarn install`
2. Start the app <br>
`yarn start`

> 💡 You can disable the TypeScript setup in Expo CLI with the environment variable `EXPO_NO_TYPESCRIPT_SETUP=1 expo start`
